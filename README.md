# FitxBuddy Backend

Welcome to the **FitxBuddy Backend** – a robust and scalable backend solution powering the FitxBuddy platform. This backend is built using modern technologies and adheres to best practices in API development, authentication, and database management.

## About

The **FitxBuddy Backend** is responsible for handling the business logic, data storage, user authentication, and third-party integrations of the FitxBuddy application. It and **Sequelize** as the ORM.
 is designed using **Node.js** and **Express**, with **PostgreSQL** as the primary database
---

## Tech Stack

- **Node.js** + **Express** – REST API framework
- **PostgreSQL** – Relational Database
- **Sequelize** – ORM for PostgreSQL
- **JWT** – Authentication
- **Nodemailer** – Email services
- **Morgan** – HTTP request logging
- **dotenv** – Environment configuration
- **CORS** – Cross-Origin Resource Sharing
- **Nodemon** – Live development reloading

### Installation & Setup
    Follow the steps below to set up and run the FitxBuddy Backend locally:

### 1. Clone the Repository

      git clone -b onboarding https://github.com/sdeitech/FitxBuddyBackend.git
      cd FitxBuddyBackend

### 2. Install Dependencies
       npm install

### 3. Start the Development Server

       npm run dev

   Your backend server will now be running on http://localhost:5000.
